Документация UI Kit

Кнопки

Используйте классы:

  • .button--primary — основная кнопка
  • .button--secondary — вторичная кнопка
  • .button--disabled — неактивная кнопка
  • .button--small / medium / large — размеры

Поля ввода

Используйте классы:

  • .input--normal — нормальное состояние
  • .input--focus — фокус
  • .input--error — ошибка
  • .input--with-icon — с иконкой

Карточки

Используйте классы:

  • .card — базовая карточка
  • .card--no-image — без изображения
  • .card--small / medium / large — размеры
